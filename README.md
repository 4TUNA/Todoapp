# Setup
* __Install required libraries__
    * ```djangorestframework = "*"```
    * ```django-cors-headers = "*"```
    * ```django-crispy-forms = "*"```
* __python_version = 3.9__

* __After Installation__
    * _Type this commands to terminal_
        * ```python manage.py runserver```
